# 04 — Lista de PRs abertas

Status: ready-for-agent

## Parent

`.scratch/skanner/PRD.md`

## What to build

Para um repo selecionado, listar as PRs abertas via Octokit: número, título, autor, branch
de origem, contagem de adições/remoções e data de atualização. A lista é atualizável.

## Acceptance criteria

- [ ] Selecionar um repo mostra suas PRs abertas com os metadados listados.
- [ ] Estado vazio explícito quando não há PRs abertas.
- [ ] Um "atualizar" refaz a busca da lista.
